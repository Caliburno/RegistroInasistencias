# RegistroInasistencias

Desktop application for tracking teacher absences at educational institutions.

## About

Group project developed for Instituto Tecnológico de Informática (ITI-UTU) as part of Bachillerato de Informática coursework. Designed to help administrative staff manage and display teacher absence records.

## Team

- **MPereira2025** - Database design and implementation
- **Barbourel30** - Frontend development
- **Me** - Project coordination and backend logic

## Features

- **User Authentication**: Account creation and login system with cédula validation
- **Teacher Management**: Register teachers with their personal information and assigned subjects
- **Subject Assignment**: Link teachers to subjects with group numbers and weekday schedules
- **Absence Tracking**: Record absence periods with automatic calculation of affected groups
- **Public Display**: Read-only window designed for casting on hall monitors to show current absences

## Tech Stack

- Java
- Swing (GUI)
- NetBeans
- MySQL

## Status

Completed academic project. Not deployed for production use, though designed with real-world application in mind.

The final implementation is simpler than initially planned due to project scope limitations set by course requirements.
